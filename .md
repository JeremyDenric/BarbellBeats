# 🚀 Quick Start Guide

## Setup (5 minutes)

```bash
# 1. Clone and setup
git clone <your-repo-url>
cd gym-music-share
chmod +x scripts/setup.sh
./scripts/setup.sh

# 2. Configure environment
# Edit backend/.env with:
#   - AWS credentials
#   - Spotify API keys

# Edit frontend/.env with:
#   - API URLs (after backend deployment)

# 3. Deploy backend
cd backend
npm run deploy

# 4. Start frontend
cd ../frontend
expo start
```

## Key Commands

### Backend
```bash
cd backend
npm run deploy          # Deploy to AWS
npm run deploy:prod     # Deploy to production
npm test                # Run tests
serverless logs -f functionName -t  # View logs
```

### Frontend
```bash
cd frontend
expo start              # Start dev server
npm run ios             # Run on iOS
npm run android         # Run on Android
npm test                # Run tests
```

### Development
```bash
# From root
npm run lint            # Lint all code
npm run format          # Format with Prettier
npm test                # Run all tests
```

## Architecture at a Glance

```
Frontend (React Native)
    ↓ HTTPS/WebSocket
API Gateway
    ↓
Lambda Functions
    ↓
DynamoDB + S3
```

## Key Files

| File | Purpose |
|------|---------|
| `backend/serverless.yml` | Infrastructure config |
| `backend/src/functions/` | Lambda handlers |
| `backend/src/lib/algorithms/` | Ranking logic |
| `frontend/src/screens/` | App screens |
| `frontend/src/services/api.ts` | API client |
| `shared/src/types/` | TypeScript types |

## Common Issues

**Backend won't deploy?**
```bash
aws configure  # Check AWS credentials
```

**Frontend can't connect?**
```bash
# Check frontend/.env has correct API_URL
# Ensure backend is deployed
```

**DynamoDB errors?**
```bash
# Check tables exist in AWS Console
# Verify IAM permissions
```
